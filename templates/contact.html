This website has been developed as part of the University College Dublin 2014 MSc Conversion Course in Computer Science. This coding project is the submission for COMP30440 Software Engineering Project 2014.This is a 20-credit module and is part of the MSc Programme.

The aim of this project is to combine all the skills so far learned in coding, webdesign, database administration built on a Linux or Windows environment. The concept was developed by Graham Harkness to be submitted in September 2014. This website is not currently available for public use. Sentiment analysis has been performed using the AlchemyAPI software under academic licence. For further details please contact info@grahamharkness.ie

The website requires a password to access. This is because the API is currently licenced for academic use only. Once the academic qualification has been awarded this site will revert to the standard free licence and the program will be made available for public use. In the mean time please contact us directly if you wish to access the program.
